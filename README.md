
# 롤 백과사전 클래식 — 비공식 아카이브

**English name: LoL Encyclopedia Classic — Unofficial Archive**

> 이 프로젝트의 제품명, 아이콘 및 복고풍 인터페이스는 현재 검토 중입니다. Riot Games의 공식 제품이나 League of Legends 클라이언트로 오인될 가능성이 있는 요소는 공개 출시 전에 변경할 예정입니다.

롤 백과사전 클래식은 League of Legends Classic 출시를 계기로, 과거 모바일 LoL 정보 앱을 사용하던 경험과 초기 시즌의 자료를 현대 Android 환경에서 다시 살펴볼 수 있도록 개발 중인 비공식 정보·커뮤니티 애플리케이션입니다.

현재 애플리케이션은 개발 및 비공개 테스트 단계입니다. 화면 구성, 제품 이름, 아이콘, 데이터 및 기능은 공개 출시 전까지 변경될 수 있습니다.

## 프로젝트를 시작한 계기

League of Legends Classic 출시 소식을 접한 뒤, 과거에 사용되던 모바일 LoL 백과사전과 같은 편리한 정보 앱을 현대 Android 환경에서 다시 이용할 수 있다면 좋겠다는 생각으로 이 프로젝트를 시작했습니다.

이 프로젝트는 과거 제3자 애플리케이션의 공식 후속작이나 공식 복원판이 아닙니다. Riot Games 또는 과거 제3자 애플리케이션 제작사로부터 승인받은 제품도 아닙니다.

과거 모바일 게임 정보 앱을 사용하던 경험에서 영감을 받아 독립적으로 개발하고 있습니다.

## 프로젝트 목표

이 프로젝트는 다음과 같은 목표를 가지고 있습니다.

- 과거 League of Legends 자료를 모바일 환경에서 편리하게 열람
- 초기 시즌의 챔피언, 아이템, 룬, 특성 및 주문 정보 보존
- League of Legends Classic 관련 정보 제공
- 현재 라이브 정보와 과거 보존 자료의 명확한 구분
- 사용자들이 과거 시즌의 추억과 정보를 공유할 수 있는 커뮤니티 제공
- 현대 Android 환경에 맞는 접근성과 사용성 개선

## 주요 기능

현재 개발 중이거나 구현을 계획하고 있는 기능은 다음과 같습니다.

### 과거 정보 아카이브

- 과거 챔피언 정보
- 챔피언 능력치 및 스킬 정보
- 챔피언 배경 이야기와 플레이 팁
- 과거 아이템 및 조합 정보
- 룬 정보 및 룬 페이지
- 공격·방어·보조 특성 정보
- 소환사 주문 정보
- 과거 시즌 관련 용어 사전

### 현재 라이브 정보

Riot Games가 공식적으로 제공하는 문서화된 API를 이용해 다음 정보를 선택적으로 제공할 예정입니다.

- Riot ID 계정 조회
- 소환사 프로필 정보
- 현재 랭크 정보
- 챔피언 숙련도
- 최근 경기 기록
- League of Legends 서비스 상태

과거 자료와 현재 라이브 정보는 별도의 화면과 안내 문구를 통해 명확하게 구분합니다.

과거 보존 자료는 현재 League of Legends Classic의 실제 데이터와 다를 수 있으며, 현재 게임 정보로 표시하지 않습니다.

## Riot Games API 사용 계획

사용을 계획하고 있는 Riot Games API는 다음과 같습니다.

- `account-v1`
- `summoner-v4`
- `league-v4`
- `champion-mastery-v4`
- `match-v5`
- `lol-status-v4`

모든 Riot Games API 요청은 별도의 HTTPS 백엔드 서버를 통해 처리할 예정입니다.

프로덕션 API 키와 서버 비밀값은 다음 위치에 포함하지 않습니다.

- Android 애플리케이션
- JavaScript 소스
- APK 또는 AAB 파일
- 공개 GitHub 저장소
- 클라이언트에서 접근할 수 있는 설정 파일

이 애플리케이션은 문서화되지 않은 Riot Games API 엔드포인트를 이용하거나, 비공식적인 방법으로 Riot Games의 데이터를 수집하지 않습니다.

개발용 및 임시 API 키는 공개 서비스에 사용하지 않습니다. 공개 서비스에는 해당 제품에 대해 검토 및 승인된 프로덕션 API 키만 사용할 예정입니다.

하나의 프로덕션 API 키를 여러 제품에서 공유하지 않습니다.

## 플레이어 정보 사용 원칙

이 프로젝트는 Riot Games API에서 제공받은 정보를 이용해 특정 플레이어를 비난하거나 부정적으로 평가하지 않습니다.

다음과 같은 기능은 제공하지 않습니다.

- 플레이어의 최근 경기력을 근거로 한 비난 또는 낙인
- 트롤 점수 또는 플레이어 평판 점수
- 비공식 MMR 또는 ELO 계산
- 공식 랭크 시스템을 대체하는 순위 시스템
- 특정 플레이어의 제재 여부를 판단하는 기능
- 게임 내 신고 체계를 대체하는 신고 기능
- 다른 플레이어를 공개적으로 평가하거나 박제하는 기능
- 게임 플레이에 불공정한 이점을 제공하는 기능

계정, 랭크 및 경기 정보는 정보 조회와 사용자의 자기 분석을 돕는 범위에서만 제공할 예정입니다.

## 커뮤니티 게시판

사용자들이 다음 내용을 공유할 수 있는 공용 커뮤니티 게시판을 제공할 예정입니다.

- League of Legends Classic 관련 이야기
- 과거 시즌의 추억
- 챔피언과 아이템에 대한 일반적인 이야기
- 일반적인 게임 정보와 공략
- 애플리케이션 개선 의견
- 오류 및 기능 제안

커뮤니티 게시판은 특정 플레이어를 평가하거나 비난하고, 게임 내 신고 체계를 대신하기 위한 목적으로 사용하지 않습니다.

다음과 같은 게시물과 행위는 허용하지 않을 예정입니다.

- Riot ID 또는 소환사명을 이용한 특정 플레이어 박제
- 특정 플레이어를 트롤, 대리 게임 이용자 또는 부정행위자로 단정하는 게시물
- 경기 기록을 이용한 플레이어 비난 또는 낙인
- 플레이어 평판 점수 및 비공식 평가
- 게임 내 신고를 대신 접수하거나 유도하는 게시물
- 개인정보 및 비공개 대화 내용의 무단 공개
- 괴롭힘, 혐오, 위협 또는 불법 콘텐츠
- 도박, 베팅 또는 금전 거래를 유도하는 콘텐츠
- 저작권이나 기타 권리를 침해하는 콘텐츠

## 커뮤니티 안전 및 운영 계획

공개 출시 전 다음 기능과 정책을 마련할 예정입니다.

- 서비스 이용약관 동의
- 개인정보처리방침
- 커뮤니티 운영정책
- 게시글 및 댓글 신고
- 사용자 신고 및 차단
- 운영자의 게시글 및 댓글 삭제
- 반복 위반 계정에 대한 이용 제한
- 부적절한 콘텐츠 관리
- 회원 탈퇴
- 사용자 데이터 삭제 요청
- 신고 내역 검토 및 처리 절차
- 운영자용 관리 및 제재 도구

커뮤니티 게시판은 지속적인 관리와 모니터링을 전제로 운영할 예정입니다.

## 현재 구현 상태

현재 Android 프로토타입에는 과거 정보 열람 기능과 기기 내부에 저장되는 시험용 게시판이 포함되어 있습니다.

현재 게시판 데이터는 각 사용자의 기기 내부에만 저장되며 다른 사용자와 공유되지 않습니다.

다음 기능은 아직 구현 전이며 공개 배포 전에 별도로 구축하고 테스트할 예정입니다.

- 공용 온라인 게시판
- 사용자 계정 및 인증
- 게시글과 댓글 동기화
- 사용자 신고 및 차단
- 운영자 관리 및 제재 기능
- 사용자 데이터 삭제 기능
- Riot Games API 중계 백엔드
- 프로덕션 API 키를 사용하는 서버 환경

아직 구현되지 않은 기능을 현재 이용 가능한 기능처럼 표시하지 않습니다.

## 전체 개발 현황

현재 진행 상황은 다음과 같습니다.

- Android 프로토타입 개발 중

- 주문 아이콘과 데이터 대응 관계 검증 중
- 인터페이스와 접근성 개선 중
- 공용 게시판 백엔드 설계 중
- Riot Games API 중계 백엔드 설계 중
- 커뮤니티 관리 기능 준비 중
- Google Play 비공개 테스트 준비 중

현재 제공되는 화면은 개발 중인 프로토타입이며 최종 공개 버전과 다를 수 있습니다.

## 데이터 구분 원칙

애플리케이션에서는 다음 두 종류의 정보를 명확하게 구분합니다.

### 과거 자료

- 과거 시즌 자료를 토대로 구성
- 자료의 기준 시기와 가능한 출처 표시
- 현재 게임 데이터라고 표시하지 않음
- 현재 League of Legends Classic과 다를 수 있음을 안내
- 정확성이 확인되지 않은 내용은 검증 필요로 표시

### 현재 라이브 정보

- Riot Games가 제공하는 문서화된 API를 통해 조회
- 조회 시점의 계정·랭크·경기 정보
- 과거 자료와 별도 화면 및 표시 사용
- API 응답 내용을 임의로 플레이어 평가 점수로 변환하지 않음

확인되지 않은 정보를 공식 정보처럼 단정하지 않으며, 검증이 필요한 자료에는 별도의 안내를 표시합니다.

## 자산 사용 원칙

공개 버전에서는 Riot Games가 공식적으로 제공하거나 서드파티 개발자에게 사용을 허용한 자산만 사용하도록 검토할 예정입니다.

가능한 경우 다음과 같이 공식적으로 문서화되거나 허용된 출처를 우선 사용합니다.

- Riot Games Data Dragon
- Riot Games Press Kit
- 공식적으로 문서화된 정적 데이터
- 사용 조건과 출처가 명확한 기타 허용 자산

Riot Games의 공식 로고는 애플리케이션 아이콘, 제품 로고 또는 홍보 이미지로 사용하지 않습니다.

Riot Games의 공식 애플리케이션이나 League of Legends 클라이언트로 오인될 수 있는 이름, 아이콘, 로고 및 화면 구성은 공개 출시 전에 수정할 예정입니다.

권리 관계를 확인할 수 없는 과거 제3자 애플리케이션의 배경, 버튼, 로고 및 기타 독자적인 그래픽은 공개 출시 전에 교체하거나 사용 권한을 별도로 확인할 예정입니다.

현재 프로토타입에 포함된 자산이 공개 배포에 적합하다는 의미는 아닙니다. 공개 출시 전 자산별 권리와 사용 조건을 다시 검토할 예정입니다.

## 브랜드 및 디자인 원칙

이 프로젝트는 Riot Games 또는 League of Legends의 공식 제품처럼 보이도록 제작하는 것을 목표로 하지 않습니다.

복고풍 인터페이스는 초기 모바일 게임 정보 앱의 사용 경험을 떠올릴 수 있도록 제작한 개발 단계의 디자인입니다.

다음 원칙을 적용할 예정입니다.

- Riot Games 공식 로고 사용 금지
- Riot Games와의 제휴·승인·후원을 암시하는 표현 금지
- 공식 League of Legends 클라이언트와 혼동될 수 있는 디자인 수정
- 제품 이름에 비공식 프로젝트라는 성격을 명확하게 표시
- 앱 시작 화면과 소개 페이지에 비공식 제품 고지 표시
- Riot Games의 검토 의견에 따라 제품 이름, 아이콘 및 화면 구성 변경

Riot Games의 공식 제품과 혼동될 가능성이 있다고 판단되는 부분은 공개 출시 전에 변경할 예정입니다.

## 수익화

현재 애플리케이션은 무료 제공을 계획하고 있습니다.

공개 출시 시점에는 다음 기능을 포함하지 않을 예정입니다.

- 유료 다운로드
- 유료 구독
- 특정 사용자만 이용할 수 있는 독점 기능
- 게임 플레이에 불공정한 이점을 제공하는 유료 기능
- 도박 또는 베팅
- 금전적 가치가 있는 자체 화폐
- 플레이어 평가 또는 신고와 관련된 유료 기능

향후 광고, 후원, 구독 또는 기타 수익화를 검토하는 경우 기능을 구현하기 전에 Riot Games 개발자 정책과 Google Play 정책을 다시 확인할 예정입니다.

필요한 경우 Riot Games Developer Portal을 통해 사전 문의하고 제품 설명과 등록 정보를 갱신한 뒤 진행할 예정입니다.
